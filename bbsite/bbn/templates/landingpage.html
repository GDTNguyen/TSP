What is the TSP?

The Travelling Salesman Problem is a well-studied problem in combinatorial optimization. The general TSP is NP hard, this means that the problem is a non-deterministic polynomial acceptable problem. In other words, it is as hard as the hardest problem in NP, this can be proved by showing that all Hamiltonian problems, which are NP, can be reduced to TSP in polynomial time.

The significance of this is that if we can find a polynomial time algorithm to get an optimal solution for this NP hard problem then there will be one for all the NP problems. Current methods allow us to search for an good solutions that is close to the optimal for large problems sets, methods such as a Backbone aide in finding an optimal solution in good time.
